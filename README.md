## 1 단계

- [x] JPA를 활용하여 데이터베이스에 접근하도록 수정한다.

- [x] `gradle spring-boot-starter-data-jpa`  의존성 추가
- [x] 엔티티 매핑
    - [x] 다른 클래스를 의존하지 않는 클래스 먼저 엔티티 설정을 한다.

- [x] 연관관계 매핑
    - [x] 다른 클래스에 의존하는 클래스는 연관관계 매핑을 추가로 한다.
